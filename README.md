# [Udemy Certificate Generator](https://certi-gen.herokuapp.com/)

This is a simple web app that generates Udemy certificates.
Created only for educational purpose.

## Prerequisites

- [NodeJS](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)

## Getting Started

### Installing packages

```sh
    yarn
```

### Running the app

```sh
    yarn dev
```
